Amplify safe or boring designs with strategic boldness — not making everything louder.

## Prerequisites

1. **Read anti-patterns before amplifying**:
   - Read `.agentctx/context/conventions/anti-patterns.md` — avoid going too far

2. Identify what to amplify:
   - If argument provided, focus on that area: $ARGUMENTS
   - Otherwise, scan for screens that feel generic, timid, or flat

## Steps

### Identify What Makes It Boring

- Generic choices: system fonts, basic colors, standard layouts
- Timid scale: everything medium-sized with no drama
- Low contrast: everything has similar visual weight
- Static: no motion, no energy, no memorable moments

### Amplify Strategically

- **Typography**: dramatic size jumps (3x-5x, not 1.5x), pair 900 weights with 200 weights, use distinctive font choices
- **Color**: increase saturation, introduce unexpected combinations, let one bold color own 60% of colored elements
- **Spatial drama**: make important elements 3-5x larger, break the grid for hero elements, use generous whitespace (100-200px gaps)
- **Motion**: staggered entrance choreography, scroll-triggered reveals, satisfying micro-interactions
- **Composition**: create clear focal points, full-bleed elements, asymmetric layouts with tension

### Pick ONE Hero Moment

- Choose a single signature moment and make it amazing
- Everything else supports the hero, not competes with it
- If everything is bold, nothing is bold

## Important

- Bold means distinctive and confident, not chaotic or garish
- Bold design must still be usable — impact without function is decoration
- Do not sacrifice readability for aesthetics; body text must remain readable
- Avoid the AI slop trap: purple gradients, glassmorphism, and neon accents are generic, not bold
